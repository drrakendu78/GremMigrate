# Privacy Policy

**GremMigrate** is a fully client-side web application. Your privacy is respected by design.

## No Data Collection

GremMigrate does not collect, store, or transmit any personal data. No accounts, no registration, no tracking.

## No Cookies

This application does not use cookies of any kind — no session cookies, no analytics cookies, no third-party cookies.

## No Network Requests

After the initial page load, GremMigrate makes **zero network requests**. Your profile files are processed entirely in your browser and never leave your device.

## File Processing

When you drop or select a Joystick Gremlin profile file:
- The file is read locally using the browser's File API
- The XML is parsed and converted entirely in JavaScript
- The converted output is generated in-memory
- No data is sent to any server

## No Analytics

There is no analytics, telemetry, or tracking of any kind. No Google Analytics, no Plausible, no Matomo — nothing.
